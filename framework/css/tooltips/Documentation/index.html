CSS3 Tooltips Pack
CSS Styles

The project uses the following cascade of style sheets:

  • page.css - style sheet that describes the page,
  • tooltips_main.css - main style sheet for tooltips
  • tooltips_styles.css - style sheet for predefined tooltip styles (skins).

These style sheets are located in styles directory.

Implementation and Usage
  • First, attach the external CSS files: tooltips_main.css and tooltips_styles.css to your web page e.g.: 
    <link rel="stylesheet" type="text/css" href="styles/tooltips_main.css"/>
<link rel="stylesheet" type="text/css" href="styles/tooltips_styles.css"/>
    You can reduce tooltips_styles.css file size by removing styles that aren't used on the page. Doing it please make sure you have an intact copy of the original files.
  • Now you can attach tooltips to the different page elements in your HTML document e.g.: 
    <a class="qlabs_tooltip_top qlabs_tooltip_style_1" href="#"><span>Tooltip Text</span>Some Text</a>
<p class="qlabs_tooltip_top qlabs_tooltip_style_1"><span>Tooltip Text</span>Some Text</p>
<h1 class="qlabs_tooltip_top qlabs_tooltip_style_1"><span>Tooltip Text</span>Some Text</h1>
<div class="qlabs_tooltip_top qlabs_tooltip_style_1"><span>Tooltip Text</span>Some Text</div>
<div class="qlabs_tooltip_top qlabs_tooltip_style_1"><span>Tooltip Text</span><img src="#" alt="image"/></div>
Image and Movie Support

Tooltips support img and iframe tag to attach images or videos. Please note that the part of code with iframe tag won't validate!

  • Tooltip with image: 
    <a class="qlabs_tooltip_top qlabs_tooltip_style_1" href="#"><span><img src="samples/image_1.jpg" alt="image_1.jpg"/>Tooltip Text</span>Some Text</a>
  • Tooltip with movie: 
    <a class="qlabs_tooltip_top qlabs_tooltip_style_1" href="#"><span><iframe width="240" height="152" src="http://www.youtube.com/embed/JknpvVHtaW8" frameborder="0" allowfullscreen></iframe>Tooltip Text</span>Some Text</a>
Form Fields Tooltips

You can apply toolips to textfields - inputs or textareas across a form with OnHover or OnFocus actions.

  • Form inputs and textareas tooltips on hover: 
    <div class="qlabs_tooltip_right qlabs_tooltip_style_1">
	<input type="text" name="" value="" />
	<span>Tooltip Text</span>
</div>

<div class="qlabs_tooltip_right qlabs_tooltip_style_1">
	<textarea name="message" cols="" rows=""></textarea>
	<span>Tooltip Text</span>
</div>
  • Form inputs and textareas tooltips on focus: 
    <div class="qlabs_tooltip_focus_right qlabs_tooltip_style_1">
	<input type="text" name="" value="" />
	<span>Tooltip Text</span>
</div>

<div class="qlabs_tooltip_focus_right qlabs_tooltip_style_1">
	<textarea name="message" cols="" rows=""></textarea>
	<span>Tooltip Text</span>
</div>
Addons
  • To add a header to the tooltip add immediately after span element: 
    <strong>Tooltip Header Text</strong>

e.g.: <a class="qlabs_tooltip_top qlabs_tooltip_style_1" href="#"><span><strong>Tooltip Header Text</strong>Tooltip Text</span>Some Text</a>
  • To add a delay to animation add class delay_100, delay_200, delay_300, delay_500 or delay_700 e.g.: 
    <a class="qlabs_tooltip_top qlabs_tooltip_style_1 delay_500" href="#"><span>Tooltip Text</span>Some Text</a>
Customization
  • To change the appearance of the tooltip use qlabs_tooltip_style_1 to qlabs_tooltip_style_50 classes. Click here for skins preview e.g.: 
    <a class="qlabs_tooltip_top qlabs_tooltip_style_25" href="#"><span>Tooltip Text</span>Some Text</a>
<a class="qlabs_tooltip_top qlabs_tooltip_style_9" href="#"><span>Tooltip Text</span>Some Text</a>
  • To change the position of the tooltip use one of 6 predefined position classes: qlabs_tooltip_top, qlabs_tooltip_bottom, qlabs_tooltip_left, qlabs_tooltip_right, qlabs_tooltip_diagonal_left or qlabs_tooltip_diagonal_right e.g.: 
    <a class="qlabs_tooltip_diagonal_right qlabs_tooltip_style_1" href="#"><span>Tooltip Text</span>Some Text</a>
<a class="qlabs_tooltip_bottom qlabs_tooltip_style_1" href="#"><span>Tooltip Text</span>Some Text</a>
  • To change the tooltip width open styles/tooltips_main.css, search for Tooltip Width and change min-width: 180px; to any other value.
  • To change the tooltip default font open styles/tooltips_main.css, search for Tooltip Base Font Family and for Tooltip Base Font Size and change font-family or font-size.
  • If you would like to change only the header text font, open styles/tooltips_styles.css and search for .qlabs_tooltip_style_1 span strong. It's the place where you can override font settings only for tooltips headers. Search for .qlabs_tooltip_style_47 span strong to get an example.
  • If you would like to change only text color, open styles/tooltips_styles.css and search for e.g.: .qlabs_tooltip_style_1 span. Here you'll find text color declaration.
